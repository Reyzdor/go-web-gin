# Go Web Application with Gin

A backend web application written in Go using the Gin framework. It implements user authentication with JWT tokens, role management, and basic CRUD operations.

# Functionality

User registration and authentication

JWT tokens (access + refresh)

Authentication checking middleware

User and session management

Post and user data models

SQLite database

# Requirements

Go 1.16 or higher

# Installation
```
git clone https://github.com/Reyzdor/go-web-gin.git
    	
cd go-web-gin
    	
go mod download
    	
go run main.go 
```

# Database

The project uses SQLite. The app.db database file is created automatically upon first launch.

Client-side project - https://github.com/Reyzdor/rest-go-gin
